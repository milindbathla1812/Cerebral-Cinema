## Related Preprocessed Datasets

The complete collection of preprocessed naturalistic fMRI datasets is available on Hugging Face:

* **Narratives:** https://huggingface.co/datasets/qazwxedc/narratives-preprocessed-fmri
* **LPP:** https://huggingface.co/datasets/qazwxedc/lpp-preprocessed-fmri
* **BOLDMoments:** https://huggingface.co/datasets/Rishav230707/BoldMoments_Preprocessed

Each repository contains the preprocessed outputs for its respective dataset, generated using the same preprocessing pipeline to provide a consistent representation for downstream neuroimaging and machine learning applications.
